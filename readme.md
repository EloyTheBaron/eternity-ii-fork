# Eternity II Puzzle Solver & Playground

Python toolkit para resolver y analizar **Eternity II puzzle** y sus variantes. Combina interfaz gráfica interactiva, múltiples algoritmos de resolución (greedy, hill climbing, backtracking, genético) y herramientas de análisis.

# Definición de Piezas

Las piezas se definen con: **ID, E (Este), S (Sur), W (Oeste), N (Norte)**

Ejemplo: `139,14,8,8,9`
- Pieza ID 139
- Borde Este = 14
- Borde Sur = 8
- Borde Oeste = 8
- Borde Norte = 9

📁 Archivo: [eternity2_256.csv](./data/eternity2/eternity2_256.csv)

# Definición de Hints (Pistas)

**Formato:** `fila, columna, id_pieza, rotación`

Ejemplo: `8,7,139,3`
- Pieza 139 va en fila 8, columna 7
- Rotación 3 = girada 270° en sentido horario

📁 Archivo: [eternity2_256_hints.csv](./data/eternity2/eternity2_256_hints.csv)

```bash

               -hints data/eternity2/eternity2_256_hints.csv
```

Controles:
- **Click izq**: seleccionar pieza para intercambio
- **Click der**: rotar pieza
- **i**: mostrar/ocultar números
- **s**: guardar solución

## 3. Resolver Automáticamente

```bash
# Terminal 1: Monitorear
python monitor.py -conf data/eternity2/eternity2_256.csv -dir ./

# Terminal 2: Resolver (elegir uno)
python replacing.py -conf data/eternity2/eternity2_256.csv -hints ...    # Greedy
python swapping.py -conf data/eternity2/eternity2_256.csv -hints ...     # Hill climbing
python backtracking.py -conf data/eternity2/eternity2_256.csv -hints ... # Fuerza bruta (lento!)
```

## 4. Mejorar Solución Existente

```bash
# Cargar y limpiar
python remove_nonmatching.py -conf puzzle.csv -load partial.csv
# → Guarda limpio como "manual_save_<uuid>_<score>.csv"

# Mejorar desde la versión limpia
python replacing.py -conf puzzle.csv -load manual_save_*.csv -hints ...
```

# 💡 Conceptos Clave

### Puntuación (Score)
Número de **matches**: bordes adyacentes con el mismo color/patrón.
- **Máximo 256×256**: 256×255 + 256×255 = **130,560 matches**

### Tipos de Pieza
- **Corner** (4): dos bordes exteriores
- **Edge** (60): un borde exterior
- **Inner** (236): sin borde exterior

### Orientación
Cada pieza puede estar girada: 0°, 90°, 180°, 270°
- Se almacena como `dir ∈ {0, 1, 2, 3}`

### Hints
Piezas pre-colocadas y rotadas correctamente (no intercambiables).

# 📞 Preguntas Frecuentes

**P: ¿Por qué es tan difícil?**  
R: Eternity II es NP-completo. Solo 2 personas lo resolvieron (6+ años combinados). Nuestro back-end lo hace más tratablecon hints + buenos algoritmos.

**P: ¿Cuál algoritmo es mejor?**  
R: Depende del puzzle. Greedy (reemplacing) bueno al inicio, hill climbing (swapping) mejor para refinamiento. Combinar ambos da mejores resultados.

**P: ¿Puedo crear puzzles propios?**  
R: Sí, usa `puzzle_generator.py` o crea archivos CSV manualmente siguiendo el formato.

**P: ¿Hay GPU support?**  
R: No actualmente. El proyecto es CPU pure. Extensiones bienvenidas.
